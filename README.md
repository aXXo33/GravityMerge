# Boom Chain

Aim, shoot & trigger epic chain reactions! An addictive puzzle game with neon visuals, satisfying explosions, boss battles & infinite levels.

## Play Now

[Play Boom Chain](https://axxo33.github.io/BoomChain/)

## Features

- Chain reaction puzzle gameplay
- Neon visuals with glass-breaking effects and particle animations
- Boss battles with shields
- Infinite procedurally generated levels
- Star rating system
- Satisfying combos and explosions
